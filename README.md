# Projeto-css-2026
Este projeto é uma aplicação web que utiliza Inteligência Artificial para gerar código HTML e CSS automaticamente a partir de uma descrição feita pelo usuário.

O usuário escreve o que deseja criar (por exemplo: “bola azul quicando”) e a aplicação envia essa descrição para uma API de IA. A IA retorna o código HTML e CSS correspondente, que é exibido na tela e também mostrado em um preview visual em tempo real.

🧩 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

HTML5 – Estrutura da aplicação

CSS3 – Estilização da interface

JavaScript – Lógica e interação da aplicação

API de Inteligência Artificial (Groq / Llama) – Geração automática de código

Google Fonts (Inter) – Tipografia da interface
